## Deployment — GitHub Pages

This site is served via **GitHub Pages** from the `main` branch root directory.

**Settings:** Repository → Settings → Pages → Source: `main` branch, `/ (root)`

The `.nojekyll` file ensures GitHub Pages serves the raw HTML without Jekyll interference.
